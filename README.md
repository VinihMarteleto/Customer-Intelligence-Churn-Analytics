# Customer Intelligence – Churn Analytics

Este projeto apresenta o desenvolvimento de um modelo de Machine Learning voltado à previsão de cancelamento de clientes (churn), com foco em geração de inteligência estratégica para retenção e redução de perdas financeiras.

O objetivo é simular um cenário real de negócio no qual empresas precisam identificar, de forma preditiva, clientes com alto risco de evasão para direcionar ações preventivas baseadas em dados.

---

## Visão Geral do Projeto

O projeto contempla um fluxo completo de análise e modelagem preditiva:

- Leitura e tratamento de dados
- Análise exploratória
- Engenharia de variáveis
- Separação entre base de treino e teste
- Treinamento de modelos supervisionados
- Avaliação de desempenho
- Interpretação dos resultados sob perspectiva de negócio

---

## Tecnologias Utilizadas

- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

As bibliotecas utilizadas permitem manipulação eficiente de dados, construção de modelos preditivos e avaliação estatística de desempenho.

## Modelagem Preditiva

Foram aplicados algoritmos de Machine Learning para classificação do risco de cancelamento, incluindo:

- Árvore de Decisão
- K-Nearest Neighbors (KNN)

Os modelos foram avaliados com base em métricas de desempenho para comparação e validação da capacidade preditiva.

---

## Objetivo de Negócio

A aplicação prática deste modelo permite que empresas:

- Identifiquem clientes com maior probabilidade de churn
- Antecipem perdas de receita
- Desenvolvam estratégias de retenção mais assertivas
- Tomem decisões orientadas por dados
